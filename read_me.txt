# WRR-Reservoir
Written by Armeen Taeb, March 2021
This code is associated with the paper titled " Perturbations and Causality in Gaussian Latent Variable Models"

NOTE: CVX-MATLAB should be installed with its folder included in the main folder. This can be downloaded from "http://cvxr.com/cvx/"

1) ``real_datasets_analysis" this contains the code and dataset for the real data experiments. The code is written in Matlab

2) ``generating_data_for_synthetic_experiments": contains codes to generate data for the synthetic experiments in the paper. This must be run first before executing item 3 below!

3) ``Synthetic_analysis": after executing 2, generates the synthetic results in the paper.

4) "supplementary_figure": to generate the figures in supplementary material
